Hashiri San Francisco

Web Design & Front-End Development

Hashiri SF is a restaurant website designed to reflect the brand's refined, contemporary identity. Excluding the logo, I led the visual direction, web design, imagery planning, and full front-end development, delivering a custom HTML experience.

View Site

Scope

  • UI/UX design
  • Visual direction
  • Image planning and photo requests
  • Front-end architecture (HTML / CSS / JavaScript)
  • Animation design and implementation
  • Responsive development
  • Performance optimization

Challenges

  • Translating a high-end dining experience into a digital format
  • Curating imagery to support atmosphere and brand tone
  • Creating subtle motion without distracting from content
  • Maintaining performance without CMS support

Approach

  • Defined layout and typographic hierarchy aligned with the restaurant’s positioning
  • Directed image selection and composition to ensure visual consistency
  • Built a fully custom HTML structure
  • Implemented restrained animation to enhance flow and pacing
  • Optimized assets to maintain fast load times

Results

The site communicates a clear and cohesive brand presence. Imagery and layout work together to reflect the restaurant's atmosphere. The implementation remains lightweight and performant across devices. The custom structure allows precise control over presentation and interaction.
