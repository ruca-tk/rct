mahola wallet

Product Design

mahola wallet is an internal NFT collection application designed to manage and interact with digital assets. I led the product design from the ground up, defining the visual identity, user flows, feature set, and interaction system beyond the original brief.

Scope

  • Product UI/UX design
  • User flow architecture
  • Feature definition & expansion
  • Gamification mechanics
  • Account system design
  • Interaction and feedback states

Challenges

  • Designing an intuitive NFT management interface for non-technical users
  • Introducing structure where functional requirements were incomplete
  • Balancing gamification with usability

Approach

  • Defined core product flows: onboarding, collection, item detail, and account management
  • Added missing essential functions including password reset, account deletion, and language switching
  • Introduced gamified elements such as status levels and achievement feedback
  • Designed clear interaction states for acquisition, usage, and history tracking
  • Built a cohesive dark UI system aligned with the Web3 positioning
  • Structured components and documented interaction states to ensure seamless handoff to web app developers

Results

The product evolved from a basic NFT viewer into a structured wallet experience with defined account management and engagement mechanics. Gamification increased clarity around progression and collection status. Critical user flows were formalized and standardized, improving usability and completeness. The interface established a consistent visual identity aligned with the broader mahola ecosystem.
